# Branding

Interested in using our brand content? Don't worry, we've got you covered!

## Logos

Do not modify (that includes cropping, resizing, changing colors, drawing on top, etc.) in any way the logos. You <u>must not</u> use our name and logo to mislead people into thinking that you are the owner of the website or that you are partnered with us (unless you are explicitly allowed by us to do so).

You can use any of the following versions of the logo:

export const Asset = ({ imageUrl, title, subtitle, dark = true }) => {
    return (
        <div className="flex flex-col gap-2">
            <Link download={true} href={imageUrl}>
                <img
                    src={imageUrl}
                    className={
                        "w-full rounded-lg p-4 " +
                        (dark ? "bg-neutral-900" : "bg-neutral-100")
                    }
                />
            </Link>
            <div>
                <div className="font-bold text">{title}</div>
                <div className="text-neutral-400 text-sm">{subtitle}</div>
            </div>
        </div>
    );
};

## Colors

Our colors are pretty simple:

export function Color({ name, color, dark }) {
    return (
        <div
            className="h-32 rounded-xl border-2 border-neutral-100 dark:border-neutral-800 flex flex-col p-4 items-end justify-end"
            style={{ backgroundColor: color, color: dark ? "white" : "black" }}
        >
            <div className="font-bold text-xl">{name}</div>
            <div>{color}</div>
        </div>
    );
}
