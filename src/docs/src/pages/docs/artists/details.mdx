# Get an Artist by ID

This endpoint allows you to get an artist by its ID.

## Endpoint Details

-   URL: `/v3/artists/{id}`
-   Method: `GET`
-   Response: `JSON`

## Parameters

| Location | Name | Type    | Description     | Required |
| -------- | ---- | ------- | --------------- | -------- |
| Path     | `id` | Integer | The artist's ID | Yes      |

## Example Request
