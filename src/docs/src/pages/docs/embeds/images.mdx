To add an embed to your website, you'll just need to copy this into your HTML:

```html showLineNumbers /:id/
<!-- Remember to replace :id with the actual image ID -->
<iframe
    src="https://api.nekosapi.com/v2/images/:id/embed"
    frameborder="0"
></iframe>
```

## Configurations

All extra configurations need to be passed as query parameters. For example, `?theme=dark`. Multiple configurations at a time are allowed.

| Query parameter | Description                                                                                                                               | Possible values          | Default    |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------- | ------------------------ | ---------- |
| `theme`         | Change the embed's theme to light or dark.                                                                                                | `light`, `dark`          | `light`    |
| `orientation`   | Set the embed's orientation to vertical (image on top, details at the bottom) or to horizontal (image on the left, details on the right). | `vertical`, `horizontal` | `vertical` |
| `palette`       | Enable or disable the image's color palette.                                                                                              | `true`, `false`          | `true`     |
| `nekos_logo`    | Adds a tiny Nekos API logo at the bottom right corner of the embed.                                                                       | `true`, `false`          | `false`    |
