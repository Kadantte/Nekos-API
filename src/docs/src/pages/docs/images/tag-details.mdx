# Get a Tag by ID

This endpoint allows you to get a tag by its ID.

## Endpoint Details

-   URL: `/v3/images/tag/{id}`
-   Method: `GET`
-   Response: `JSON`

## Parameters

| Location | Name | Type    | Description   | Required |
| -------- | ---- | ------- | ------------- | -------- |
| Path     | `id` | Integer | The tag's ID. | Yes      |

## Example Request

<Tabs items={["Python 3", "JavaScript"]}>
    <Tabs.Tab>
        ```python filename="tag_details.py" showLineNumbers

        res = requests.get(f"https://api.nekosapi.com/v3/images/tag/{tag_id}")
        ```
