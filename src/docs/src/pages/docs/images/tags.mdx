# Get an Image's Tags

This endpoint allows you to get an image's tags.

## Endpoint Details

-   URL: `/v3/images/{id}/tags`
-   Method: `GET`
-   Response: `JSON`

## Parameters

| Location | Name     | Type             | Description                   | Required |
| -------- | -------- | ---------------- | ----------------------------- | -------- |
| Path     | `id`     | Integer          | The image's ID.               | Yes      |
| Query    | `limit`  | Integer [1..100] | The amount of tags to return. | No       |
| Query    | `offset` | Integer          | The amount of tags to skip.   | No       |
